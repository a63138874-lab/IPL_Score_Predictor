#IPL Score Predictor

An end-to-end Machine Learning project that predicts the final score of an IPL match based on live match conditions such as overs, wickets, runs, and other match features. This project includes a trained ML model and a simple UI for user interaction.

📌 Project Overview

This project uses historical IPL data (2008–2024) to train a regression model that can predict the final score of a team during a live match scenario.

It helps users understand:

How match conditions affect scoring
Real-time score prediction using ML
Basic deployment with UI (Streamlit / Gradio / Tkinter)
⚙️ Features
📊 Data preprocessing on IPL dataset (2008–2024)
🤖 Machine Learning model training
🧠 Score prediction based on live inputs
🖥️ Simple UI for interaction
💾 Model saved using Pickle
📈 Visual insights (optional graphs)
